Пример использования функции мультиподписания через HTTP API KAZTOKEN mobile/desktop библиотеки https://github.com/sigex-kz/ncalayer-js-client

Работает только с KAZTOKEN mobile/desktop, функиция мультиподписания через HTTP API KAZTOKEN mobile/desktop позволяет подписывать неограниченные количества документов любых размеров.

Выберите файлы для подписания, нажмите Отправить на подпись, отобразится интерфейс KAZTOKEN mobile/desktop. После выбора хранилища и ключа будут вычислены подписи и отображены на странице.
